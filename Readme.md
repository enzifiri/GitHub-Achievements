## üöÄ Step-by-Step Guides: How to Get Them

Below are detailed instructions for specific badges. You can research the others on the internet!

<details>
<summary><b>1. Pull Shark Achievement</b></summary>

**Requirement:** Create 2 Pull Requests and have them merged.

1. **Fork** this repository from the top right.
2. Go to your profile, click on **Repositories**, and enter your forked `Github-Basarimlar` repo.
3. Click on the **pencil icon** to edit the file.
   ![Edit File](https://user-images.githubusercontent.com/76253089/206911370-11792ad3-9289-4719-b2db-e8ed7774288d.png)
4. Add a simple period (`.`) at the end of a sentence.
5. Scroll down to **Commit changes**, select `Create a new branch...`, and click **Propose changes**.
6. Click **Create Pull Request**.
7. Go to [this link](https://github.com/kriptodostu/Github-Basarimlar/compare/) to send me the PR. 
   * **Important:** Ensure the base repository is `kriptodostu/Github-Basarimlar` (base: `main`) and the compare branch is yours.
8. Click **Create Pull Request** again. 
9. **Repeat this process 2 times.** Once I approve them, your badges will appear. *(Contact on TG: @enzifiri)*
</details>

<details>
<summary><b>2. Quickdraw Achievement</b></summary>

**Requirement:** Open and close a pull request within 5 minutes.

1. **Fork** this repo just like you did for the Pull Shark achievement and edit the file.
2. Create the Pull Request.
3. Immediately go to the opened Pull Request page, scroll to the bottom, and click **Close pull request**. That's it!
   ![Close PR](https://user-images.githubusercontent.com/76253089/206918372-765a550e-6aba-43f5-9f0d-95595cda51cd.png)
</details>

<details>
<summary><b>3. Pair Extraordinaire Achievement</b></summary>

**Requirement:** Co-author a commit.

1. **Fork** this repo and open the file for editing (pencil icon).
2. Make a small change (like adding a period).
3. Scroll down to **Commit changes**, select `Create a new branch...`, and in the **description/comment box**, paste the following exact code:

   ```text
   Co-authored-by: Kral001 <kral16024@gmail.com>
   Co-authored-by: @kriptodostu <kriptodostu@gmail.com>
Click Propose changes and then Create Pull Request.

Make sure you set the base to kriptodostu/Github-Basarimlar just like in the first step, and finalize the PR.

Wait for approval. (Any questions? Tag @enzifiri in Rues Community chat)

</details>

<details>
<summary><b>4. Galaxy Brain Achievement</b></summary>

Requirement: Have 2 of your discussion answers marked as "accepted".

Go to our Discussions Page.

Find a question that does not have a green checkmark next to it (green means it's already answered). If there are no open questions, message @enzifiri on Telegram to create one.

Type any helpful text and click Comment.

Important: You must answer 2 different questions. You cannot answer the same question twice.

Wait for the question owner to approve your answer.

</details>

<details>
<summary><b>5. YOLO Achievement</b></summary>

Requirement: Merge a pull request in your own repo without a review.

Go to your GitHub profile and create a New Repository.

Go to your repo's Settings > Collaborators and add my email so I can participate:

Plaintext
kriptodostu@gmail.com
Go back to your repo's main code page, click the pencil icon to edit the README.md, make a small change, and create a Pull Request.

On the right side of the PR page, look for the Reviewers section and assign me.

Because it is your repo, ignore the review and directly click Merge pull request.
(Note: If I am not in the reviewers section, I can't verify it. Send your GitHub username to @enzifiri on TG if needed).

</details>

<summary><b>6. Public Sponsor Achievement (Paid: $1)</b></summary>

Requirement: Sponsor an open-source project.

Go to the curl GitHub Sponsors page.

Fill in your payment information and confirm by clicking Sponsor curl. Your badge will appear on your profile shortly!
